# SalesFlow Dashboard

A modern business analytics dashboard with real-time metrics, revenue tracking, and order management.

## Features

- **Revenue Analytics** — Interactive charts showing daily/weekly/monthly trends
- **Order Management** — Track orders with status filtering (completed, pending, cancelled)
- **Customer Insights** — Traffic source breakdown and customer growth metrics
- **Responsive Design** — Works on desktop, tablet, and mobile
- **Dark Mode** — Clean dark UI designed for extended use

## Tech Stack

- HTML5 / CSS3
- Vanilla JavaScript
- SVG Charts (no external dependencies)
- Responsive CSS Grid & Flexbox

## Getting Started

```bash
git clone https://github.com/favexus/SalesFlow-Dashboard.git
cd SalesFlow-Dashboard
# Open index.html in your browser, or:
python -m http.server 3000
```

Then visit `http://localhost:3000`
